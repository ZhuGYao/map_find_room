# 仿链家地图找房

### 技术交流群：1018373095

### 主要功能点
    1. 以北京为基础，区->街道->小区,三级下钻联动。
    2. 根据地图视野动态加载相应点位，以此来提高性能。
    3. 画圈找房，根据在地图上随意画的区域来找到区域内的房源。
    4. 具体效果以及功能，请下载源码直接运行便可观看，另外注释写的很清楚。
